All Longitundinal Bleeds

Longitudinal days are based on a bleed date, relative to a new medication start, within the following parameters:

  • Bleed date between 0 to -180 days (-6 months) BEFORE a med start
  • Farthest longitudinal follow-up date of 540 days (18 months) AFTER a med start
